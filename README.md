# Fake-News-Detector

**Tools used:**

* Python
* Tensorflow
* TFID, CountVectorizer
* Flask, HTML, CSS

**How to use:**

The way to use it, is very simple.

1. Execute `python web.py`
2. On a browser, insert the ip that is shown in the terminal: `http://XXX.XXX.XXX.XXX:5000` or `http://localhost:5000`
